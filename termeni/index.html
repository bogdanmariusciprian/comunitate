Termeni de utilizare

Regulile casei — pe scurt, clar și pe înțelesul tuturor.

1. Despre platformă

Atelierul-LRO este o platformă educațională pentru limba și literatura română (gimnaziu și liceu), administrată de profesorul coordonator. Conținutul lecțiilor are scop educațional.

2. Contul și conduita

Comunitatea este destinată elevilor. Ne purtăm frumos: limbajul vulgar este filtrat automat, iar conținutul semnalat este verificat de profesor. Mesajele dintre membri folosesc șabloane predefinite; mesaje libere se pot trimite doar profesorului.

3. Conținutul tău

Postările, comentariile și exercițiile propuse îți aparțin. Publicându-le, permiți platformei să le afișeze în comunitate. Îți poți șterge oricând propriul conținut; profesorul poate modera orice conținut care încalcă regulile.

4. Limitări

Platforma este oferită „ca atare”, în scop educațional. Punctele, insignele și clasamentul sunt elemente de joc, fără valoare monetară.
